Flyer

This was the first major assignment for my Digital Game Design course, which tasked us with making a simple toy in Perlenspiel. This meant that there could be no rules or goals. I chose to make simple toy that entailed flyer around in the air and crashing into other planes.

An initial presentation was done to show the class my idea for the toy, alongside conceptual drawings and sample code. These presentation slides can be found here. I also did a final presentation showing the final outcome of the project, complete with screenshots and snippets of the final code. These slides can be found here.

Most of the programming was done by me in Javascript. The professor of the course, who also created the engine, helped with some programming, mainly in order to show me how the engine handled things such as sprites. All code changes made by him that were kept in the final version are marked with a comment that starts with *BM*. The audio came from the Perlenspiel Audio Library. I created the sprites and the cover art.

The toy can be played online here. The toy's web page can be inspected in order to see the main script: "game.js" At the top of this script are credits to everyone who helped with development.

This is one of my projects from a WPI course called Digital Game Design, which uses the Perlenspiel engine. This engine uses Javascript and restricts the user to creating games on a grid of beads. Each of these games are playable on Edge, Chrome, and Firefox, but they run best on Chrome. More information about the engine can be found here
