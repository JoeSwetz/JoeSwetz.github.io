Flyer

This was the first major assignment for my Digital Game Design course, which tasked us with making a simple toy in Perlenspiel. This meant that there could be no rules or goals.

The game's web page can be inspected in order to see the game script: "game.js" At the top of this script are credits to everyone who helped with development.

Most of the programming was done by me in Javascript. The professor of the course, who also created the engine, helped with some programming, mainly in order to show me how the engine handled things such as sprites. All code changes made by him that were kept in the final version are marked with a comment that starts with *BM*.

The toy can be played online here: http://users.wpi.edu/~jsswetz/IMGD-2900/A07/A07%20(Toy)/cover.html

This is one of my projects from a WPI course called Digital Game Design, which uses the Perlenspiel engine. This engine uses Javascript and restricts the user to creating games on a grid of beads. Each of these games are playable on Edge, Chrome, and Firefox, but they run best on Chrome. More information about the engine can be found at: http://users.wpi.edu/~bmoriarty/ps/index.html
