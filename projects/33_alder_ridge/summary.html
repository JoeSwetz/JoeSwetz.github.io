33 Alder Ridge

This game was created as the final project for a class at WPI titled "Advanced Storytelling: Quest Logic and Level Design." We were tasked with creating some sort of level in Unreal Engine 4, and my partner and I decided we wanted to make a creepy house, inspired by PT. We took it a step further and decided to turn it into a small game where you can inspect objects around the house to learn about what happened at this house. This video above is the final video submission for the class, where we talk through the level.

I created the game with a partner, Anne Higgins, whose website can be found here. I did the programming for the project, and Anne helped find and create art assets for the level. The voice acting was done by Jim Plante and Cameron Person.

From an educational perspective, I wanted to learn how to create an object inspection mechanic for both the sake of the level, and also to understand how to do it for future games, as it is a common game mechanic. I learned how to do this by following this tutorial.

The game is available to download here.
