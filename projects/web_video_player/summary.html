Web Video Player

This was a group project that I worked on for my Software Engineering class. I worked in a group of 4 to create the application using Java, Amazon Web Services and an SQL database. The application allows participant users to upload and manage short video clips in their own personal library, as well as create playlists of those videos. They can also make videos publicly available and gain access to videos in other people's libraries. Administrator users can add/remove sites to gain access to the videos from those sites, as well as mark which of their own videos are publicly available.

Personally, I worked on a portion of the controller functions for the use cases that were laid out at the beginning of the term. Some of these functions also interacted with the SQL database, which kept track of the videos, playlists and remote sites for the user. I also wrote test cases for those functions, and managed the groups Amazon Web Services account and API for the application.
