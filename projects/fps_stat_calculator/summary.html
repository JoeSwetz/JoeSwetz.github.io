Current work: FPS Stat Calculator

I'm currently in a class called Webware, where we're learning about web development through the development of full-stack applications using Node.js. The current project I'm working on is a First Person Shooter stat calculator, where a user can enter and track their stats from different rounds of an FPS game, and the application will calculate kill/death ratio and assist/ratio and keep track of totals and averages for kills, assists and deaths. The user can also download all this data as a CSV file for easy generation of graphs and charts in a program like Microsoft Excel.

The back-end server uses Express and a MongoDB database. The front-end uses the Bulma CSS framework.
